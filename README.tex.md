# Higher-order primes / Super-primes

Countably list all primes:

$$p_1, p_2, p_3, p_4, p_5, \ldots$$

such that, for all $i$:

1. $p_i$ is prime.
2. $p_i < p_{i+1}$.
3. There is no prime $q$ such that $p_i < q < p_{i+1}$.
